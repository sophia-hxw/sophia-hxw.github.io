CAT-Enhancing Multimodal Large Language Model to Answer Questions in Dynamic Audio-Visual Scenarios

发表于 2024-11-21 分类于 UNDO Valine：
本文字数： 367 阅读时长 ≈ 1 分钟

前言

主要是面向AVQA(audio visual question and answer)和AVC(audio visual caption)问题，方法自然是MLLMs(multimodal large language models)，文章的出发点则是先有方法推理结果的二义性，无法描述音视频中的特定事件。

本文方法：

  • 1，设计了一个clue收集策略，用推理时的问题在音视频场景中的clue来丰富LLM模型可以接受到的信息；
  • 2，模型在混合多模态数据集上训练，可直接用于类似场景，本团队也整理了AVinstruct多模态数据集；
  • 3，提出了AI辅助下的二义性DPO(direct preference optimization)算法，是消除二义性的模型重训练策略。
